perfmjs
=======

high performance javascript framework

特性:
1.fast in default: 高效，易用，易读
2.jQuery是该框架的可选部分，dom操作,ajax,css等可以用jquery语法，也可以不用
3.核心库压缩后只有62k，还可以优化至更小
4.原生态支持面向对象(OO)功能
5.原生态支持lazyload,bigRender等高效功能
6.更高效的js和css的资源加载功能，有参考head.js的实现
7.更高效的资源文件版本号管理功能
8.支持沙箱，日志等功能
9.可测试性
